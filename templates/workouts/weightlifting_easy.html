Easy Weightlifting Workouts

1) The total time required to complete this workout is 1 hour.

Start off by stretching.

Start off by warming up using the bar. Perform 3 sets of 8-12 reps of bench press. Adjust weight so that each set can be completed with at least 8 reps. Take a rest of 3 minutes between sets Move to should press machine. Perform 3 sets of 10-12 reps of shoulder press. Adjust weight accordingly. Take a 2 minute rest between sets Perform 3 sets of 8-10 reps of tricep pull downs. Adjust weight so that each set goes until failure. Rest 2 minutes between each set.
