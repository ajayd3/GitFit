Moderate Weightlifting Workouts

1)The total time required to complete this workout is 1 hour, 15 minutes

Start off by stretching.

1. Start off by warming up using the bar. Perform 4 sets of 8-10 reps of bench press. Adjust weight so that each set can be completed with at least 8 reps. Take a rest of 3 minutes between sets 2. Move to the shoulder press machine. Perform 4 sets of 8-12 reps of shoulder press. Adjust weight accordingly. Take a 2 minute rest between sets 3. Perform 3 sets of 8-10 reps of incline bench press. Adjust weight to go till failure on each set. 4. Perform 4 sets of 8-10 reps of tricep pull downs. Adjust weight so that each set goes until failure. Rest 2 minutes between each set. 5. Perform 3 sets of 8-10 reps of skull crushers. Adjust weight so that each set goes till failure.
